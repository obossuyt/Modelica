Buildings.Electrical.Transmission.LowVoltageCables

Package of low voltage electricity cables used in distribution grid

Information

This package contains records of physical properties for low voltage commercial cables. New cables can be added by extending the base record Buildings.Electrical.Transmission.LowVoltageCables.Generic.

For low voltage cables, only the characteristic resistance and reactance are specified. See Buildings.Electrical.Transmission.Base.BaseCable for a comprehensive list of parameters that can be specified for a cable.

Extends from Modelica.Icons.MaterialPropertiesPackage (Icon for package containing property classes).

Buildings.Electrical.Transmission.LowVoltageCables.Generic

Data record for a generic low voltage cable

Information

This is a base record for specifying physical properties for low voltage commercial cables. New cables can be added by extending the it.

For low voltage cables, only the characteristic resistance and reactance are specified.

Extends from Modelica.Icons.MaterialProperty (Icon for property classes), Buildings.Electrical.Transmission.BaseClasses.BaseCable (Record that contains the properties of a generic cable).

Parameters

TypeNameDefaultDescription
MaterialmaterialBuildings.Electrical.Transmi...Material of the cable
CurrentAmp0.0Ampacitance of the cable [A]
TemperatureT_ref298.15Reference cable temperature [K]
TemperatureM228.1 + 273.15Temperature constant of the material [K]
replaceable function lineResistancelineResistanceFunction that computes the resistance of a cable
replaceable function lineInductancelineInductanceFunction that computes the inductance of a cable
replaceable function lineCapacitancelineCapacitanceFunction that computes the capacitance of a cable
CharacteristicResistanceRCha Characteristic resistance of the cable [Ohm/m]
CharacteristicReactanceXCha Characteristic reactance of the cable [Ohm/m]

Modelica definition

record Generic "Data record for a generic low voltage cable" extends Modelica.Icons.MaterialProperty; extends Buildings.Electrical.Transmission.BaseClasses.BaseCable; parameter Buildings.Electrical.Types.CharacteristicResistance RCha(start=0) "Characteristic resistance of the cable"; parameter Buildings.Electrical.Types.CharacteristicReactance XCha(start=0) "Characteristic reactance of the cable"; redeclare function extends lineResistance "Function that computes the resistance of a cable" input Buildings.Electrical.Transmission.LowVoltageCables.Generic cable "Record that contains cable properties"; algorithm R :=cable.RCha*l; end lineResistance; redeclare function extends lineInductance "Function that computes the resistance of a cable" input Buildings.Electrical.Transmission.LowVoltageCables.Generic cable "Record that contains cable properties"; algorithm L := l*(cable.XCha/omega); end lineInductance; redeclare function extends lineCapacitance "Function that computes the capacitance of a cable" input Buildings.Electrical.Transmission.LowVoltageCables.Generic cable "Record that contains cable properties"; algorithm C := 0.0; end lineCapacitance; end Generic;
