Buildings.Fluid.SolarCollectors.Data.Concentrating.C_CogenraSunDeck

C - Cogenra Solar, Inc., SunDeck 1.0

Information

Necessary data for calculating dp_nominal, G_nominal and dT_nominal is not provided in the EN12975 test data. All are left blank.

References

Ratings data taken from the Solar Rating and Certification Corporation website. SRCC# = 2011127A.

Buildings.Fluid.SolarCollectors.Data.Concentrating.C_VerificationModel

C - All inputs necessary for verification of EN12975 models

Information

No model on the Solar Rating and Certification Corporation website tested to EN12975 standards provides all of the necessary information for modeling. Specific limitations are dp_nominal, G_nominal and dT_nominal. This data record was created to allow verification of EN12975 base classes despite the limitations in available data.
