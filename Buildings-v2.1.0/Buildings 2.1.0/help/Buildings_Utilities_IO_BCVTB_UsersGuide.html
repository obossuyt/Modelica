User's Guide

Information

This package contains an interface to the Building Controls Virtual Test Bed (BCVTB). The BCVTB is a middleware that is based on Ptolemy II. Ptolemy II is a software framework to study modeling, simulation, and design of concurrent, real-time, embedded systems, with focus on the assembly of concurrent components and the use of heterogeneous mixtures of models of computation that govern the interactions between components. The BCVTB adds functionalities to Ptolemy II that allows coupling Modelica, EnergyPlus and MATLAB/Simulink for data exchange during the time step integration.
To run the examples in Buildings.Utilities.IO.BCVTB.Examples the BCVTB needs to be installed. The BCVTB installation contains these examples and the C libraries that are required by this package.
