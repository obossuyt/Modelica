Buildings.Electrical.AC.OnePhase

One phase AC systems

Information

Package with models for alternate current (AC) single phase systems.
The models contained in this package use the phase system Buildings.Electrical.PhaseSystems.OnePhase and they assume 120 V as nominal RMS voltage.
