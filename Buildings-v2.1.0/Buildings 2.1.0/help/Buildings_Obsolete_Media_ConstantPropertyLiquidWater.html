Buildings.Obsolete.Media.ConstantPropertyLiquidWater

Package with model for liquid water with constant properties

Information

This medium model is identical to Modelica.Media.Water.ConstantPropertyLiquidWater, except that it implements the function that computes the specific internal energy.

Extends from Buildings.Obsolete.Media.Interfaces.PartialSimpleMedium (Medium model with linear dependency of u, h from temperature. Most other quantities are constant.).
